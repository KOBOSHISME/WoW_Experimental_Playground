Place addon-local custom sound files here.

Use lower-case file names without spaces for simple slash-command testing, for example:

```text
/wep tools sound play custom:wep-alert.wav duration=1 channel=sfx
```

WoW addons commonly use `.ogg`, `.mp3`, or `.wav` files. These files do not need to be listed in the `.toc`; they just need to be installed with the addon folder.
